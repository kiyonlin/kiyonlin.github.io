# 迭代器
一个迭代器对象 ，知道如何每次访问集合中的一项， 并记录它的当前在序列中所在的位置。在  `JavaScript` 中 迭代器是一个`对象`，它提供了一个 `next()` 方法，返回序列中的下一项。这个方法返回包含`done`和`value`两个属性的对象。
